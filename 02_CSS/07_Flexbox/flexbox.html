Flexbox Masterclass Demo

Flexbox, short for Flexible Box Layout, is a CSS layout module designed to help create flexible and responsive layouts. It allows you to arrange and align items within a container efficiently, even when their sizes are dynamic or unknown.

1. Default Flex (row)

Flexbox default property is row.

3. Justify Content: Space-Between

The justify-content property is used to align the flex items when they do not use all available space on the main-axis(horizontally).It always works left to right axis(main axis). Property Values: center, flex-start, flex-end, space-around, space-between, space-evenly.

4. Align Items: Center

The align-items property is used to align the flex items when they do not use all available space on the cross-axis(vertically).It always works top to bottom.Anything start with align will work top to bottom. axis. Property Values: center, flex-start, flex-end, stretch, baseline, normal.

5. Flex Wrap: Wrap

The flex-wrap property specifies whether the flex items should wrap or not, if there is not enough room(space) for them on one flex line. Property Values: nowrap, wrap, wrap-reverse.

The align-content property is used to align the flex lines.The align-content property is similar to align-items, but instead of aligning flex items, it aligns the flex lines. Property Values: center, stretch, flex-start, flex-end, space-around, space-between, space-evenly.

7. Order Property

The CSS order property is used to control the order of flex or grid items within their container. It allows you to change the visual order of items without affecting their source order in the HTML document. Property Values: 0(default), Positive Number, Negative Number.

8. Flex Grow

The flex-grow property in CSS defines how much a flex item will grow relative to the other items inside a flex container. It determines how the available space in the container is distributed among the flex items. Property Values: 0(default), Positive Number, Negative Number not allowed.

9. Align Self

The align-self property in CSS allows you to control the alignment of an individual flex or grid item along the cross axis (perpendicular to the main axis). It overrides the align-items property for that specific item. Property Values: auto(default), flex-start, flex-end, center, baseline, stretch.
