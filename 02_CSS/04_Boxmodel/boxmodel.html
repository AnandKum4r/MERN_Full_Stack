CSS Box Model Example

Box-Sizing

CSS provides the box-sizing property to control how the box model behaves. There are two main values:
1-content-box(default)
2-border-box

1-Content-box (Default)

The content box is the area where your content (text, images, etc.) is displayed.By default, the width and height properties you specify in CSS only apply to the content box.

This is a content-box element.

Width: 300px + 40px (padding) + 10px (border) = 350px

Height: 200px + 40px (padding) + 10px (border) = 250px

Here, width and height apply only to the content box. Padding and border are added outside the specified dimensions. box.

2-Border-box

This is a border-box element.

Width: 300px (including padding and border)

Height: 200px (including padding and border)

Here, width and height include the content, padding, and border. Makes it easier to control element sizes without manual calculations.

Inline vs Block Elements

This is a paragraph with an inline element inside. Inline elements only take up as much width as necessary and allow other elements next to them.

This is a block-level element. It takes up the full width available and creates a new line before and after it.

Box Model Components

The box model consists of:

  • Content: The actual content of the box, where text and images appear
  • Padding: Clears an area around the content (inside the border)
  • Border: A border that goes around the padding and content
  • Margin: Clears an area outside the border

Total Width = Content Width + Padding (left + right) + Border (left + right) + Margin (left + right) Total Height = Content Height + Padding (top + bottom) + Border (top + bottom) + Margin (top + bottom)
